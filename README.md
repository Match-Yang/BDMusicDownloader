## BDMusicDownloader
=================

一个Chrome插件，可以用来下载百度无损音乐，支持Chrome浏览器和UC桌面浏览器，其他Webkit内核的浏览器应该也支持，但是我没测试。
src文件夹下是源代码，Linux与Mac系统下可以直接装src.crx文件，Windows系统需要下载BDMusicDownloader.zip文件，以开发者模式加载。

## 原理
